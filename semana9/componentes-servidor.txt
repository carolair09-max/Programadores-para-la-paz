Sistema operativo:
Es el software principal que administra los recursos del computador o servidor. 
Permite ejecutar programas, gestionar archivos y controlar el hardware.

Node.js:
Es un entorno de ejecución que permite ejecutar código JavaScript fuera del navegador. 
Se utiliza para crear servidores y aplicaciones web.

Aplicación server.js:
Es el archivo donde se define la lógica del servidor.
 Su función es recibir solicitudes de los usuarios y enviar las respuestas correspondientes.

Puerto:
Es un punto de comunicación que permite al servidor recibir conexiones.
 Cada aplicación utiliza un número de puerto específico para funcionar.

Logs:
Son registros que almacenan información sobre la actividad y los eventos del servidor. 
Ayudan a detectar errores, monitorear el sistema y solucionar problemas.
