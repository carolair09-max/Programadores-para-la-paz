El login es el proceso mediante el cual un usuario se identifica en una aplicación usando 
sus credenciales, como un nombre de usuario y una contraseña. Esto permite verificar su 
identidad antes de acceder al sistema. Algunas rutas deben estar protegidas para que solo 
los usuarios autorizados puedan consultar o modificar información importante. De esta 
forma, se evita el acceso no autorizado y se protege la seguridad de la plataforma. No deben
exponerse públicamente datos personales, contraseñas, documentos de identidad, direcciones,
teléfonos, claves de acceso ni otra información sensible.
